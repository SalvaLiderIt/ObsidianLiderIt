

Para buscar un fragmento de código de entre todas las ramas a la vez, colocar el cursor en la raíz del repositorio y utilizar el comando
```bash
git grep "¿Confirma que desea cambiar Fact. a-Nº contacto?" $(git rev-list --all)

```
La parte de verde es la que se utiliza para poner el fragmento de código deseado

Para salir de esta "ventana" dentro de la terminal darle a la tecla de salir (q)
